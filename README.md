# Image to Wbox Conversion



## How To Use



1. Get an image
2. Move the image into **inputs** folder
3. Boot up the program using either **run\_windows.bat** or **run\_linux.sh**
4. Enter the image file name (with the file extension) when prompted
5. Enter the desired name of your output file
6. Wait for it to say "Done!"
7. *Boom*, you've got a worldbox map
8. Check [here](https://the-official-worldbox-wiki.fandom.com/wiki/Maps#Map_Installation) for information on how to install WorldBox maps



### Requirements

The only requirement is **Java**.



### Supported Image File Types

* BMP
* JPEG
* PNG
* TIFF
* WBMP



### Additional Use Information

Each pixel in the image equates to one tile. To make the world smaller or larger, resize the image. Worldbox worlds can only be in 64 tile x 64 tile chuncks. Images who's measurements are not divisible by 64 will be cropped. (Top side and right side will be cropped out.) Really big maps may crash your game. Nearest color is based on CIEDE2000 algorithm.
